Создаю нового бота в тг @BotFather.

Создаю пустой проект в webstorm.
Затем запускаю команду npm init -y (создаётся package.json).

**** ДОБАВЛЯЮ ПРОЕКТ В GIT ****
Чтобы добавить проект в гит создаю там новый репозиторий.
Копирую команду "git remote add origin https://github.com/troll-gnyot-el/scoliologic-bot.git"
Создаю файл .gitignore , в который пишу папки, которые не нужно загружать в гит
Проинициализируем проект "git init"

** Удаление файла из GIT **
git rm --cached README.txt
git commit -m "удалён README.txt file"
git push origin master

## Краткая логика структуры
Пользователь шаг за шагом выбирает
конечность → уровень ампутации → интересующий вопрос → получает персонализированный ответ (видео/инфо).
